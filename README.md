## Install
``` bash
npm install f2f
```

## Usage
```js
var f2f = require('f2f');
var F2F = new f2f();

F2F.simplef2f('ثقلین'); // Outputs: "saghlin"
F2F.simplef2f('قسطنطنیه'); // Outputs: "ghastntnih"
```

```html

  var F2F = new f2f();
  alert(F2F.simplef2f("قسطنطنیه"));

```

#### F2F.simplef2f([text])

##### text
Type: `string`

String in Persian to be converted.
